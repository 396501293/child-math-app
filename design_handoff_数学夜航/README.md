# Handoff: 数学夜航 — 5岁儿童 iPad 数学学习 App（探险夜航方向）

## Overview
面向 5 岁儿童的 iPad 横屏数学学习 App。核心循环：关卡地图（章节内蛇形 15 关）→ 每关 5 道题（20 以内加减 + 缺数题）→ 星级结算 → 解锁下一关。带几何风吉祥物、语音朗读（设计中以 🔊 标注位置）、可交互计数块教具。

## About the Design Files
本包内文件是 **HTML 设计参考**（原型），展示预期外观与行为，**不是可直接搬运的生产代码**。任务是在目标代码库的既有环境（SwiftUI / React Native / Flutter / Web 等）中，用其既有模式与库**重新实现**这些设计；若尚无代码库，请为「儿童 iPad 教育应用」选择最合适的技术栈（推荐 SwiftUI 或 React Native）再实现。

## Fidelity
- `数学夜航原型.dc.html` — **高保真可交互原型**：颜色、字号、间距、交互逻辑均为最终意图，应像素级还原（在 1024×768 逻辑画布上等比缩放适配实际屏幕）。
- `数学App线框图.dc.html` — 探索过程存档（线框 + 3 个视觉方向），仅供背景参考；最终方向为其中 **3b 探险夜航**。

### 2. 答题（quiz）
- **Layout**：同背景。
  - 顶栏：返回键 68×68 圆角 20 `rgba(255,255,255,.1)`；进度条（高 16、圆角、底 `rgba(255,255,255,.12)`、填充 `#F2A541`、宽度动画 .4s）；`n/5` 计数 24px/700；🔊 重播键 68×68 `rgba(242,165,65,.2)`。
  - 题目行（top:120 居中）：96px/900 白字，运算符与等号 `#F2A541`；未知数为 120×120 虚线框（6px dashed `#F2A541`、圆角 26、底 `rgba(242,165,65,.08)`、内 `?` 70px）。两种排版：`a op b = ?` 与缺数题 `a + ? = c`。
  - 计数块面板（top:300 居中）：容器 `rgba(255,255,255,.07)` 圆角 20，块 56×56 圆角 12：
    - 减法：a 个 `#2FB29B` 块，点击→`rgba(232,93,93,.25)` 底 + `✕`（#E85D5D 30px/900），可再点还原。
    - 加法：a 个 `#2FB29B` + b 个 `#F2A541` 两组，点击变暗（opacity .55）辅助点数。
    - 缺数：a 个实心 `#2FB29B` + (c−a) 个虚线空槽（3px dashed `rgba(242,165,65,.7)`），点空槽填充为 `#F2A541`。
    - 面板下方提示行 19px `rgba(255,255,255,.5)`，以 🔊 开头（= 该题语音文案，见 Interactions）。
  - 答案区（bottom:56 居中，间距 36）：3 个选项 150×110、圆角 26、`rgba(255,255,255,.1)` 底、3px `rgba(255,255,255,.25)` 边框、52px/900 白字。
- **交互**：
  - 答对：该选项变 `#2FB29B`；全屏遮罩 `rgba(14,40,49,.55)` + 居中卡片（`#2FB29B` 底、圆角 32、56px/900「太棒了！ ★」、pop 动画 .35s：scale .6→1.08→1）；1.1s 后进下一题或结算。
  - 答错：选项变 `rgba(232,93,93,.3)` + `#E85D5D` 边框 + 横向 shake .4s，随后保持 60% 透明（排除项）；遮罩显示 `#E85D5D`「再试一次！」0.9s；不扣分不打断，可继续作答。错误次数计入星级。

## Interactions & Behavior
- **出题规则**（每关 5 题，可配 3–10）：第 1–5 关 10 以内加法；6–10 关 20 以内加/减混合（无负数）；11–15 关缺数题 `a + ? = c`（c ≤ 20）。选项 = 正确答案 ±2 内取 3 个去重乱序。
  - ⚠️ 本条已被细化替代，以 [`题库难度与模式设计.md`](./题库难度与模式设计.md) 为准（主线扩为 3 章 × 15 关 = 45 关、目标年龄 4–7 岁、约束生成器、干扰项规则、无尽/限时模式）。原文保留仅作历史参考。
- **星级**：整关答错 0 次 = 3 星；≤2 次 = 2 星；否则 1 星。取历史最高保存。
- **解锁**：完成第 n 关解锁 n+1（上限 15）。
- **语音（TTS/配音）**：所有 🔊 位置需真实语音——进地图欢迎语、点题自动朗读题目（数字重读）、计数块提示（如「13 个方块，划掉 5 个，剩下几个？」）、答对/答错反馈、结算祝贺。5 岁儿童识字有限，语音是主要引导通道。
- **适配**：以 1024×768 设计基准，`scale = min(vw/1024, vh/768)` 整体等比缩放居中；仅横屏。
- **儿童易用性**：全部点击目标 ≥ 56px（缩放前）；无惩罚机制；答错仅提示重试。

## State Management
- `screen: 'map' | 'quiz' | 'result'`
- `progress: { stars: {level: 0-3}, unlocked: number }` — **持久化**（原型用 localStorage key `math_nightsail_v1`；正式版用本地存储/云同步）
- 关内状态：`questions[]、qIndex、wrongTotal、wrongThis、pickedWrong[]、blockStates[]、feedback`
- 可配置项（原型 Tweaks）：`hardMode`（全部缺数题）、`questionCount`（3–10，默认 5）、`showBlocks`（是否显示计数块教具）

- 颜色：背景渐变 `#12333E→#174552`；面板 `rgba(255,255,255,.06/.07/.1/.12)`；主强调（琥珀）`#F2A541`；行动/当前（珊瑚红）`#E85D5D`；教具/正确（青绿）`#2FB29B`；深字 `#12333E`；白字层级 `#fff / .85 / .6 / .55 / .5 / .4`

- 动画：pulse 1.6s（当前关）、shake .4s（答错）、pop .35s（反馈）、floaty 3s（吉祥物）
